# Pritam Kayet - Sci-Fi Portfolio Website

An immersive, futuristic sci-fi themed portfolio website for Pritam Kayet, a Computer Science Engineering student at SRM Institute of Science and Technology.

## Features

- **Sci-Fi Theme**: Dark space-inspired design with neon colors and glowing effects
- **Interactive Background**: Animated particles, twinkling stars, and floating elements
- **Glass Morphism**: Modern frosted glass effects with backdrop blur
- **Responsive Design**: Mobile-friendly layout with smooth animations
- **Dark/Light Theme Toggle**: Switch between sci-fi dark and light modes
- **Particle Effects**: Floating particles that respond to mouse movement
- **Typing Animation**: Dynamic text animation on the hero section
- **Hover Effects**: Enhanced 3D transforms and glow effects
- **Smooth Scrolling**: Seamless navigation between sections

## Technologies Used

- HTML5
- CSS3 (with advanced animations and effects)
- JavaScript (Vanilla, for interactive effects)

## Project Structure

```
src/
├── index.html      # Main HTML file
├── styles.css      # CSS stylesheets
└── script.js       # JavaScript functionality
```

## How to View

Simply open `src/index.html` in your web browser.

## Sci-Fi Features

- **Background Effects**: Twinkling stars, floating particles, and gradient overlays
- **Neon Glow**: Cyan, green, and red neon colors throughout the design
- **Glass Cards**: Semi-transparent cards with blur effects
- **Particle Animation**: 50+ floating particles with random movement
- **Mouse Interaction**: Particles respond to cursor movement
- **Typing Effect**: Hero title appears with typing animation
- **3D Hover Effects**: Cards lift and rotate on hover
- **Ripple Clicks**: Click anywhere to see ripple effects

## Customization

- Update personal information in `index.html`
- Modify styles in `styles.css`
- Add functionality in `script.js`
